# Source Code V1 - Original Implementation

This directory contains the original source code for the CGM Federated Learning Benchmark.

## Files

- **`data_processing.py`** - Data processing and feature engineering (all 42 features)
- **`fl_benchmark.py`** - Federated learning benchmark runner
- **`strategies.py`** - FL strategies (FedAvg, FedProx, FedSGD, FedClusterProxXAI)
- **`generate_explanations.py`** - XAI explanation generation (SHAP/LIME)
- **`utils.py`** - Utility functions (metrics, plotting, etc.)

## Features

- Original implementation with all 42 features
- Standard data preprocessing
- Full feature set models
- Baseline federated learning strategies

## Usage

See `scripts_v1/run_experiment.py` for running V1 experiments.
